Moving on from the Synchrounous File Loop, we now look at what two do when there are 2 or more input file. Basically, we just use multple loops. One key difference, is that instead of simply sending the input directly to the output, we'll be creating and populating some data structures.

- for line in format file:
- write the line to template string, replacing data with string format parameters
- for line in data file:
- add data and data identification tags (keys) to dict
- merge the dictionary to the template, write to output file

New concepts: data structure implementation, dicts
New methods: str % dict, str.split(), str.join()*

*Despite being a string operation, this is actually to join arrays. E.g., To comma-join some array the syntax is new_array = ','.join(array)
