Building on the basic structure of the Synchrounous File Loop, we can now intergrate a multi-step line procesor to perform some numerical manipulation. For number manipulation, we need to convert, or 'cast' the string into decimal, or 'floating point' number. There is a cast function for each desired data type, e.g. str(), int() and in this case float(). Since We are going to pass strings to the float() function, the line needs to be cleaned up and split.

- 1) input/output file open()
- 2) for line in file:
- strip/split/cast/arithmetic/join/write()
- 3) file close()

New methods: str.rstrip(), str.split(), str.join()*
