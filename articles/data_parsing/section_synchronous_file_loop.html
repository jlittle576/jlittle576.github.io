Synchronous File Loop (SFL)


As seen in the Find & Replace example, a synchronous file loop loops through input and output file (or other data source) simultaneously.

Example: Find & Replace + Expand

Process a file, replacing certain text strings


SFL's capabilities exten quite a bit beyond the ability to do find and replace. Building up on top of it, we can create a class sutiable for takes which consists primarily of takeing a line of text apart. It won't take long to step beyond this scope (by storing variables into structures and exploring more comples process structures) but line de/reconstruction is a foundational set of techniques in datafile parsing.
