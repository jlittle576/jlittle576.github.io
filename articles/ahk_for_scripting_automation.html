AHK for Script Automation


I use a program called AutoHotKey to do string replacement to handle many of my common/tedious operation in various API language I have to use. Its a backround windows scripting program that can automate several windows tasks by monitoring and automating mouse and keyboard input.

On thing I use it for, is automating laborious pieces cmd language, to print (append to msg window) I type 'qma' .That is instantly erased and replaced by this:
Code:
 'int fie set fie = .gui.msg_box.message act = append str = '',

Efficiency improved by 5000%!
Even more impressive result if I type qfi ('for in')

Code:
for var = $_self.loopR sta = 1 inc = 1 end = (ROWS())
  var set var = $_self.loopObj obj = (eval( )[(eval( $_self.loopR))])

I start all of my hotstrings with 'q' to avoid strings of letters I might actually want to type. The next letters are like initials ( so qma is 'message append')
